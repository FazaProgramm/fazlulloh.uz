  • HTML

    HTML (HyperText Markup Language) — bu veb-sahifalarni yaratish va tuzishda ishlatiladigan asosiy til. HTML, veb-brauzerlarda matn, rasm, video, havolalar va boshqa turli elementlarni to'g'ri ko'rsatish uchun belgilar yordamida tuzilgan ko'rsatmalardan (markup) iborat bo'ladi.

  • CSS

    CSS (Cascading Style Sheets) — bu veb-sahifalarning ko'rinishini, dizaynini va tartibini belgilash uchun ishlatiladigan til. HTML orqali veb-sahifaning strukturasini yaratganimizdan so'ng, CSS yordamida uning ko'rinishini yaxshilaymiz. CSS sahifadagi elementlarning rangini, o'lchamini, joylashishini va boshqa uslublarini o'zgartirish imkonini beradi.

  • SASS

    SASS (Syntactically Awesome Stylesheets) — bu CSS-ni kengaytirish uchun mo‘ljallangan CSS preprocessor. U CSS kodlarini samarali va tezroq yozishga yordam beradi, kodni modulga bo‘lish imkonini beradi va takrorlanishni kamaytiradi. SASS ko‘p ishlab chiquvchilar tomonidan sevimli vosita hisoblanadi, chunki u katta loyihalar uchun qulay bo‘lgan qo‘shimcha imkoniyatlarni taqdim etadi.

  • Bootstrap

    Bootstrap — bu HTML, CSS va JavaScript yordamida tez va samarali veb-saytlar va veb-ilovalarni yaratishga yordam beruvchi mashhur Frontend framework. Bootstrap dastlab Twitter kompaniyasi tomonidan ishlab chiqilgan va ochiq kodli bo‘lib, bugungi kunda veb-ishlab chiquvchilar orasida juda ommalashgan.

  • GIT

    Git — bu ochiq kodli versiya boshqaruv tizimi bo‘lib, dasturiy ta'minotni ishlab chiqishda kodning turli versiyalarini boshqarishga yordam beradi. U jamoa bilan ishlashda juda foydali bo‘lib, bir nechta ishlab chiquvchilar tomonidan yozilgan kodlarni birlashtirish, kuzatish va boshqarish imkonini beradi.

  • Figma

    Figma — bu bulutga asoslangan dizayn va prototiplash vositasi bo‘lib, UI/UX dizayn loyihalarini yaratish, jamoaviy ishlash va interfeyslarni rejalashtirish uchun keng foydalaniladi.
